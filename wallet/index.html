🔐 Backup & Security simpan private key sebelum terlambat

⬇ Download Backup File

Download file .json berisi private key XMSS Anda. Simpan di tempat aman — USB drive, password manager, atau cloud terenkripsi.

⚠ Siapapun yang punya file ini bisa spend koin Anda.

👁 Lihat Private Key

Tampilkan XMSS secret key dan public key dalam format hex. Gunakan untuk import ke node lain dengan importxmsskey.

⚠ Jangan screenshot di tempat umum. Jangan kirim ke siapapun.
💡 Watch address dari VirtualBox? Private key ada di mesin Anda sendiri. Backup dengan: cp -r ~/.snti_mainnet/wallets/ ~/backup_wallet/ atau: bitcoin-cli dumpwallet ~/snti_backup.txt
